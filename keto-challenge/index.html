✅ المسموح في الكيتو
🥩 اللحوم والبروتين
✓ لحم بقر، خروف، دجاج، ديك رومي
✓ سمك ومأكولات بحرية بجميع أنواعها
✓ بيض كامل (أبيض وأصفر)
✓ لحم مقدد وسجق (بدون حشو نشوي)
✓ مأكولات البحر: روبيان، كالماري، بطلينوس
🥑 الدهون الصحية
✓ أفوكادو وزيت الأفوكادو
✓ زيت زيتون بكر ممتاز
✓ زبدة وسمنة
✓ زيت جوز الهند وزيت MCT
✓ دهون الحيوان (شحم، زبدة مصفاة)
🧀 الألبان الكاملة
✓ جبن شيدر، موزاريلا، بري
✓ كريمة طازجة (هيفي كريم)
✓ جبن كريمي، ماسكاربوني
✓ قشطة حامضة كاملة الدسم
✓ زبادي يوناني كامل الدسم (بكمية)
🥦 الخضار فوق الأرض
✓ سبانخ، كيل، ليتوس، جرجير
✓ بروكلي، قرنبيط، كوسا، خيار
✓ فلفل، ثوم (بكمية)، بصل (بكمية)
✓ هليون، فطر، باذنجان
✓ أفوكادو (فاكهة لكنه خضار كيتو)
🥜 المكسرات والبذور
✓ لوز، جوز، جوز برازيلي، بقان
✓ بذور شيا، كتان، قنب، يقطين
✓ زبدة اللوز والفول السوداني الطبيعية
✓ ماكاديميا (الأعلى دهوناً والأقل كرباً)
✓ بذور عباد الشمس (بكمية)
🍹 المشروبات
✓ ماء، ماء فوار، ماء معدني
✓ قهوة بدون سكر (مع كريمة)
✓ شاي أعشاب بدون سكر
✓ مرق عظام (ممتاز للكيتو فلو)
✓ ماء ليمون بدون سكر
❌ الممنوع في الكيتو
🍞 الحبوب والنشويات
✕ خبز بجميع أنواعه (أبيض، أسمر، توست)
✕ أرز، مكرونة، شعيرية
✕ دقيق قمح، ذرة، شوفان
✕ بسكويت، كعك، فطائر
✕ رقائق الذرة والبطاطس (شيبس)
🍬 السكريات والحلويات
✕ سكر أبيض وبني وعسل وشيرة
✕ عصائر الفاكهة ومشروبات غازية
✕ شوكولاتة الحليب والحلوى
✕ آيسكريم وكاسترد وبودينغ
✕ مربى، مارميلاد، صوص حلو
🍌 الفاكهة العالية بالسكر
✕ موز، عنب، بطيخ، شمام
✕ مانجو، أناناس، كرز، تين
✕ فراولة وتوت (بكميات كبيرة)
✕ عصير أي فاكهة
✕ فواكه مجففة (زبيب، تمر، مشمش)
🥔 خضار تحت الأرض
✕ بطاطس وبطاطا حلوة
✕ جزر (بكميات كبيرة)
✕ شوندر (بنجر)
✕ لفت وفجل (بكميات كبيرة)
✕ ذرة (تُصنَّف خضاراً لكنها نشوية)
🍺 الكحول والمشروبات
✕ بيرة (عالية الكربوهيدرات جداً)
✕ عصائر مُعلبة ومشروبات طاقة
✕ مشروبات رياضية محلاة
✕ قهوة سكر وشاي بالسكر
✕ حليب البقر (كميات كبيرة)
⚠️ منتجات تبدو صحية لكنها ممنوعة
✕ عسل النحل والشيرة الطبيعية
✕ خبز الحبوب الكاملة
✕ حليب اللوز المحلى
✕ زبادي قليل الدسم (يحتوي سكراً)
✕ صوص الكاتشب وكثير من التوابل الجاهزة
🔄 البدائل الكيتونية
دقيق القمح→دقيق اللوز / دقيق جوز الهند
السكر الأبيض→إريثريتول / ستيفيا / مونك فروت
الأرز→أرز القرنبيط (قرنبيط مبشور)
المكرونة→كوسا مقطعة / شعيرية الكونجاك
خبز التوست→خبز اللوز / خبز الجبن
البطاطس المقلية→فطر / قرنبيط مقلي / فلفل
الحليب العادي→كريمة ثقيلة مخففة / حليب جوز الهند
الشوكولاتة→شوكولاتة داكنة 85%+ (بكمية)
